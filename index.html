        Integração Webhook / API
      

      

        Envie requisições HTTP POST para decodificar strings Base64 remotamente. O servidor retornará o arquivo binário correspondente diretamente na resposta com os cabeçalhos apropriados.
      

      
      

        
          http://localhost:3000/webhook
        
        
      


      
      

        Ver exemplo de requisição (cURL)
        

          # Opção A: Enviando como Texto Puro (Raw Body)

          curl -X POST -H "Content-Type: text/plain" -d "JVBERi0xLjQK..." http://localhost:3000/webhook -o arquivo.pdf


          # Opção B: Enviando como JSON

          curl -X POST -H "Content-Type: application/json" -d "{\"base64\": \"JVBERi0x...\"}" http://localhost:3000/webhook -o arquivo.pdf
